Solar System 3D — Космический симулятор с метеоритной атакой
Интерактивная 3D-симуляция Солнечной системы с возможностью уничтожать планеты с помощью комет. Проект выполнен на Java с использованием JavaFX.

Возможности:
Реалистичная Солнечная система — Солнце и 8 планет с текстурами
Свободное перемещение — WASD + Q/E для движения в космосе
Вращение камеры — зажмите правую кнопку мыши и двигайте
Метеоритная атака — нажмите 1–8, чтобы запустить комету в планету или нажмите 0 для запуска кометы в солнце
Физика разрушения — планеты разлетаются на сотни осколков
Красивые орбиты — светящиеся точки, образующие орбиты планет
Звездное небо — тысячи мерцающих звезд на заднем плане
Перезапуск системы — клавиша X восстанавливает всё заново

Управление:
Правая кнопка мыши + движение - Осмотр по сторонам
W / A / S / D - Движение вперед/влево/назад/вправо
Q / E - Движение вверх/вниз
Пробел - Пауза / запуск анимации
R - Сбросить положение камеры
X - Перезапустить всю систему
1 – 8 - Запустить комету в планету
0 - Запустить комету в Солнце

Как запустить:

Требования
Java 21 или выше
JavaFX 17+
Maven (опционально)

Запуск через Maven
mvn -> plugins -> java.fx -> javafx.run
